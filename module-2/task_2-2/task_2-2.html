1. Where should you use constants, and where should you use variables? Can they all be constants?

    2. for values that do not change, use constants. For values that may change, use variables.

  2. How do parentheses affect the arithmetic outcome of an expression?

    3. Parentheses change the order of operations. Operations inside parentheses are performed first.

  3. How do we limit floating-point numbers to a given number of decimals?

    4. We can use the toFixed() method to limit floating-point numbers to a specified number of decimal places.

  4. How can we compare two floating-point values to see if they are equal (is 12.199999 the same as 12.2)?

    5. We can compare two floating-point values by checking if the absolute difference between them is less than a small threshold value (epsilon).

  5. What do comparison operators return, and how can we check if they are FALSE or TRUE?

    6. Comparison operators return boolean values (true or false). We can check their values directly in conditional statements or by printing them out.

  6. How does the Math.random() function work, and what can it be used for?

    7. Math.random() generates a pseudo-random floating-point number between 0 (inclusive) and 1 (exclusive). It can be used for simulations, games, or any scenario requiring randomization.

  7. What is modulus (%), and when do we need this operator in mathematical operations?

    8. The modulus operator (%) returns the remainder of a division operation. It is useful for determining if a number is even or odd, cycling through a set of values, or wrapping around indices in circular data structures.
